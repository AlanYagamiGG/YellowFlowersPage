Esto no es un proyecto muy serio.

Es solo una pequeña página que hice para decirte que me alegra mucho que seamos amigos,
que este año contigo ha valido muchísimo, y que aunque no pude darte flores amarillas de verdad,
al menos aquí florecen cada vez que las abres.

Las animaciones, el texto que aparece poco a poco, la música de fondo...
todo lo pensé para ti. Espero que te haya gustado.

---

### Lo que hay aquí adentro

```
index.html   — la página principal
styles.css   — los colores amarillos y el fondo suave
flowers.css  — toda la magia de las flores animadas
script.js    — la máquina de escribir y las transiciones
```

No necesita instalación, no necesita internet (bueno, solo para las fuentes).
Solo abre el `index.html` y ya.
